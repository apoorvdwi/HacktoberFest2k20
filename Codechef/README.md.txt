# CONTENTS
This directory contains solutions to various problems and their explanations.

The directory is further divided into sub-directories which have the names of the various type of problems present on codechef.

# CONTRIBUTION

1. If you want to add a solution to a problem then check if that type of subdirectory is present or not. 
2. Make a new sub-directory if a sub-directory of your type of problem is not present, also make a README file for the same(refer to already made README.md in other directories)
3. Add your solution according to the type of problem in the respective subdirectory following the rules written in that sub-directory(README). 


NOTE: EXPLANATION IS A MUST PART OF A SOLUTION.
