## CONTENTS

1. This folder contains solutions to problems named by their problem code as shown on codechef.
2. Every text document contains solutions to the problem with their explanations.

## CONTRIBUTION

Add the solution to any problem on codechef while following the undermentioned rules:  

1. Name the text document the problem code followed by the language used of the problem that you have solved on codechef.
e.g : `TREE2_C++`

2. Solution that you are adding must be totally accepted by the codechef compiler.

3. Add the code(solution) first and then write the explanation (explaining it in brief).

4. Add comments in the code(solution) 
e.g :`x+y; // This adds two numbers` in C++
